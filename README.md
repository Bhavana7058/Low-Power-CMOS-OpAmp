Low Power CMOS Op-Amp for IoT and Biomedical Applications



\## Overview

This project presents the design and simulation of a low power CMOS Operational Amplifier using Cadence Virtuoso with 180nm CMOS technology.



The Op-Amp is designed for:

\- IoT Applications

\- Biomedical Signal Processing

\- Low Power Systems



\---



\## Technology Used

\- CMOS 180nm Technology



\---



\## Features

\- Low power consumption

\- Two-stage CMOS Op-Amp

\- Stable frequency response

\- Low noise performance

\- Low voltage operation



\---



\## Tools Used

\- Cadence Virtuoso

\- CMOS 180nm Technology
